# API Endpoints

This document outlines the available API endpoints for the Newlife Website project. The base URL for all API endpoints is `/api`.

## Get Reminders for Today

-   **Endpoint:** `/get-reminders-today`
-   **Method:** `GET`
-   **Description:** This endpoint is intended to retrieve reminders scheduled for the current day.
-   **Response:** The current implementation dumps the week number and day number to the output. This endpoint is likely under development.

*Note: This file should be updated as more API endpoints are added to `routes/api.php`.*
