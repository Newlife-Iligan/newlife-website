# Filament Admin Panel

The Newlife Website project uses [Filament](https://filamentphp.com/) to provide a beautiful and powerful admin panel.

## Accessing the Admin Panel

The admin panel can be accessed by navigating to `/admin` in your browser. You will need to log in with a user account that has been granted access.

## Available Resources

The following resources are available in the admin panel. Each resource corresponds to a model in the application and allows you to perform CRUD (Create, Read, Update, Delete) operations.

-   **InventoryResource**: Manage the church's inventory.
-   **LifeGroupResource**: Manage the life groups.
-   **MemberRoleResource**: Manage the roles that can be assigned to members.
-   **MembersResource**: Manage the church members.
-   **MinistryResource**: Manage the church ministries.
-   **NlAccountResource**: Manage the Newlife financial accounts.
-   **NlFinanceResource**: Manage the Newlife financial records.
-   **RemindersResource**: Manage reminders.

Each of these resources provides a user-friendly interface for managing the corresponding data in the database.
