# Eloquent Models

This document provides an overview of the Eloquent models used in the Newlife Website project.

-   **`Inventory`**: Represents items in the church's inventory.
-   **`LifeGroup`**: Represents the small groups within the church.
-   **`MemberRole`**: Represents the different roles a member can have within the church (e.g., Pastor, Elder, Member).
-   **`Members`**: Represents the members of the church. This model likely stores personal information about each member.
-   **`Ministry`**: Represents the various ministries within the church (e.g., Youth, Music, Outreach).
-   **`NlAccount`**: Represents financial accounts for the Newlife organization.
-   **`NlFinance`**: Represents financial records or transactions for the Newlife organization.
-   **`Reminders`**: Represents reminders for tasks or events.
-   **`User`**: Represents the application's users, who can log in and interact with the system. This is the default Laravel User model.
