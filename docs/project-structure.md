# Project Structure

This document outlines the directory structure of the Newlife Website project.

-   `app/`: Contains the core code of the application.
    -   `Filament/`: Contains the Filament Admin Panel resources. Each resource corresponds to a model and defines the admin interface for it.
    -   `Http/Controllers/`: Contains the controllers that handle HTTP requests.
    -   `Models/`: Contains the Eloquent models, which represent the application's data and interact with the database.
    -   `Providers/`: Contains the service providers for the application.
-   `bootstrap/`: Contains the application's bootstrapping scripts.
-   `config/`: Contains the application's configuration files.
-   `database/`: Contains the database migrations, seeders, and factories.
    -   `factories/`: Contains model factories for generating test data.
    -   `migrations/`: Contains the database migration files.
    -   `seeders/`: Contains the database seeder files.
-   `public/`: The web server's document root. Contains the `index.php` entry file and publicly accessible assets (CSS, JS, images).
-   `resources/`: Contains the application's views, raw assets (CSS, JS), and language files.
    -   `css/`: Contains CSS source files.
    -   `js/`: Contains JavaScript source files.
    -   `views/`: Contains the application's Blade templates.
-   `routes/`: Contains the application's route definitions.
    -   `web.php`: Defines routes for the web interface.
    -   `api.php`: Defines routes for the API.
-   `storage/`: Contains compiled Blade templates, file-based sessions, file caches, and other files generated by the framework.
-   `tests/`: Contains the application's automated tests.
-   `vendor/`: Contains the Composer dependencies.

## Key Files

-   `.env`: The environment configuration file (should be created from `.env.example`).
-   `composer.json`: Lists the PHP dependencies.
-   `package.json`: Lists the JavaScript dependencies.
-   `artisan`: The command-line interface included with Laravel.
-   `vite.config.js`: Configuration file for Vite, the frontend build tool.
